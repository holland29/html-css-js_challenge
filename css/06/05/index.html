** 위치 속성 - html 태그의 박스 모델 성격에 따라, 배치 가능
-> 블록(줄 바꿈), 인라인(수평), 인라인-블록

(1) position 속성
- 좌표값에 따라 배치
  • static 요소의 기본흐름 배치
  • relative (기본흐름) 좌표 속성 사용가능
  • absolute (기본흐름x) 절대적 좌표 위치에 따라 배치
  • fixed (기본흐름x) 절대적 좌표 위치 배치, *스크롤해도 고정됨
  • sticky (static 유사 기본흐름), 지정 좌표의 임계점에 이르면 화면에 고정됨

- position: static; -> 위치 속성의 기본값 -> 화면 변화x -> 좌표 속성 사용(x) - 위치 속성을 사용하지 않은 상태와 동일함
