- 선택자: css 속성을 적용할 대상 선택

  1. 전체-selector(*)
  2. tag-selector
  3. id-selector
  4. class-selector
  5. 기본 속성-selector
  6. 문자열 속성-selector

  1. 그룹 선택자: 여러 선택자를 , 기호로 구분하여 그룹화
  2. 자식 선택자: 자식관계 범위로, >기호 사용
  3. 하위 선택자: 자식 및 자손관계 범위, 공백으로 구분
  4. 인접 형제 선택자
    - 특정 태그의 가장 인접한 형제관계
    - + 기호 구분
  5. 일반 형제 선택자
    - 특정 태그의 형제 관계의 모든 태그를 선택자로 택
    - ~ 기호를 구분자로 사용

가상 요소 선택자

- 가상으로 존재하는 선택자를 택 (실존x)
- ::(클론 2개) 가상 요소 앞에 붙임
ex.
  • ::before: 요소의 맨 앞
  • ::after: 요소의 맨 뒤

가상 클래스 선택자

- 요소의 상태를 이용해 선택자를 지정하는 방법
- :(클론 1개) 가상 클래스 선택자 앞에 붙임
  1. 링크
    • :link
    • :visited
  2. 동적
    • :hover
    • :active
  3. 입력 요소
    • :focus
    • :checked
    • :disabled
    • :enbled
  4. 구조적
    • 맏이 :fitst-child
      막내 :last-child
    • n번째 자식의 끝 :nth-child(n)
      끝에서 n번째 자식 :nth-last-child(n)
    • n번째 특정 자식 :nth-of-type(n)
      끝에서 n번째 특정 자식 :nth-last-of-type(n)
    • 부모의 맏이의 특정 자식 :fist-of-type 부모의 막내의 특정 자식 :last-of-type
